# @permifyjs/express

Express middleware adapter for PermifyJS.

> Beta warning
>
> `@permifyjs/express` is still in beta and is not recommended for production use yet unless explicitly stated otherwise.

## Install

```bash
npm install @permifyjs/core @permifyjs/express express
```
