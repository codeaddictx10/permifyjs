# @permifyjs/mongoose

Mongoose resolver adapter for PermifyJS.

> Beta warning
>
> `@permifyjs/mongoose` is still in beta and is not recommended for production use yet unless explicitly stated otherwise.

## Install

```bash
npm install @permifyjs/core @permifyjs/mongoose mongoose
```
