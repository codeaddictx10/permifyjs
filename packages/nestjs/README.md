# @permifyjs/nestjs

NestJS guard and decorator adapter for PermifyJS.

> Beta warning
>
> `@permifyjs/nestjs` is still in beta and is not recommended for production use yet unless explicitly stated otherwise.

## Install

```bash
npm install @permifyjs/core @permifyjs/nestjs @nestjs/common @nestjs/core reflect-metadata rxjs
```
