# @permifyjs/core

Framework-agnostic roles and permissions engine for JavaScript and TypeScript applications.

> Beta warning
>
> `@permifyjs/core` is still in beta and is not recommended for production use yet unless explicitly stated otherwise.

## Install

```bash
npm install @permifyjs/core
```

## What it provides

- Permission checks with `can()`
- Role checks with `hasRole()`
- Wildcard permission matching
- Assignment methods for roles and permissions
- Pluggable caching
- CLI scaffolding
