# @permifyjs/prisma

Prisma resolver adapter for PermifyJS.

> Beta warning
>
> `@permifyjs/prisma` is still in beta and is not recommended for production use yet unless explicitly stated otherwise.

## Install

```bash
npm install @permifyjs/core @permifyjs/prisma @prisma/client
```
